# Vault

Use this skill when creating or editing pages in `~/workspace/vault`.

## New pages

- Do **not** add an `# Title` heading at the top of new pages. Obsidian
  displays the filename as the page title, so a leading H1 is redundant.
- Start the file body directly with content or a frontmatter block.
- Use `##` as the highest heading level within the page body.
