Gather evidence and sources for a writing project. Run the `research` skill
for full guidance.

$ARGUMENTS is a topic, thesis, or question to research. Examples:

- `UK housing data quality`
- `whether Moore's Law has ended for software`
- `historical examples of state capacity through measurement`
- `packet switching history for the packet-service piece`

If $ARGUMENTS is blank, check whether there's a writing project in the current
conversation context and research for that.

Steps:

1. Parse the brief. What's the question? What kind of evidence is needed?
2. Search the vault for existing notes on the topic.
3. Search cass for prior sessions where this came up.
4. Search the web for primary and secondary sources, data, and
   counterarguments.
5. Fetch and verify each promising source.
6. Write a structured research note to the vault (see the skill for format
   and file location).
7. Summarise what you found: top 3-5 findings, any notable gaps, and where
   the research note lives.
